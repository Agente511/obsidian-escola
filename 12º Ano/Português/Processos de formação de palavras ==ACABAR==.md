## Processos irregulares
- ### Extensão semântica
	Uma **palavra existente** adquire um **novo significado**.
	>[!Note] Exemplos
	>- rato (animal) / rato (de computador)
	>- salvar (de um perigo) / salvar (registar dados num suporte informático)
- ### Empréstimo
	Transferência de **uma palavra de uma língua para outra**.
	>[!Note] Exemplos
	>- *croissant*
	>- *airbag*
	>- *internet*
	>- *overbooking*
	>- *Facebook*
- ### Amálgama
	**Junção de partes de duas ou mais palavras**.
	>[!Note] Exemplos
	>- informática (**infor**mação + auto**mática**)
	>- cibernauta (**ciber**nética + astro**nauta**)
	>- setora (**se**nhora + dou**tora**)
- ### Sigla
	Redução de um conjunto de palavras às suas **iniciais**, que **se pronunciam separadamente**.
	>[!Note] Exemplos
	>- GNR
	>- IRS
- ### Acrónimo
	Junção da **letra inicial ou de letras iniciais** de um conjunto de palavras, que **se pronunciam como uma palavra só**.
	>[!Note] Exemplos
	>- INEM
	>- APAV
- ### Truncação
	**Redução ou apagamento de palavras mais extensas** de que **resultam outras de menor extensão**.
	>[!Note] Exemplos
	>- metro (metropolitano)
	>- otorrino (otorrinolaringologista)
	>- foto (fotografia)
	>- pneu (pneumático)
	>- quilo (quilograma)
